## Typecho主题Sunny的开源版本

Releases可能不及时，可直接打包下载。

### 仓库地址

https://github.com/ScarletDor/typecho_theme_sunny

### 演示站点

http://sunnyfree.mmbkz.cn/

Sunny主题在2023年9月15日正式发布，此版本为发布之后的第一版。后因大量内容更新，因此后推出了更多功能的正式版。因此版本功能完整，灵活性高，因此在github放出开源版本，供各位小伙伴下载使用。

![](https://raw.githubusercontent.com/ScarletDor/typecho_theme_sunny/master/screenshot.png)


## 主题的优势

- 不夸样式了，演示站有目共睹，双栏、简洁、美观。
- 主题支持昼夜双色，支持自动昼夜切换，以及用户cookie记忆。
- 多样式的图片封面展示，优秀的动态样式风格。
- 完美适配手机端。
- ajax文章列表无刷新加载。
- 带有返回顶部，文章过期、更新提醒等多种基础功能。
- 适配Links插件，一键开启友链页面。
- 对话式的评论区样式，更加与众不同。
- 自定义字体、自定义主题色、自定义favicon等多种自定义功能。
- 数十项编辑器功能，支持网易云音乐，bilibili视频插入。

本主题有清晰合理的主题结构，比如去掉侧栏、为所有卡片添加边框阴影、均可一步实现，且无任何冲突。

## 附加说明

主题在php8.2基础上开发，理论支持php7.3-8.2

建议使用typecho1.2.1版本
